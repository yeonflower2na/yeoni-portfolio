한국소비자원 리뉴얼 프로젝트

한국소비자원(Korea Consumer Agency)은 소비자 권익 보호와 공정한 거래 환경 조성을 목표로 활동하는 공공기관입니다.
본 프로젝트는 노후된 UI를 개선하고 정보 접근성과 사용성을 높이기 위한 리뉴얼 작업입니다.
UX 설계부터 디자인, 퍼블리싱, 프론트엔드 개발까지 전 과정을 단독 수행하였습니다.

주요 기능

메인 페이지

  • 공지사항 슬라이더, 주요 배너, 팝업 기능
  • 직관적인 메뉴 구성, 모바일 대응 반응형 UI

서브 페이지

  • 피해 구제/정책 자료 등 카테고리별 정리
  • 콘텐츠 가독성 중심의 정보 구조 설계

사용 기술 및 도구

  • HTML5, CSS3, JavaScript, jQuery
  • Swiper, 반응형 미디어쿼리
  • 디자인: Figma, Photoshop
  • 개발 환경: VS Code, Windows10

디자인 방향성

  • 정보 구조 단순화: 사용자가 필요한 정보를 빠르게 찾을 수 있도록 콘텐츠를 카테고리별로 정리
  • 디자인 시스템 적용: 컬러, 폰트, 컴포넌트의 일관된 기준을 정의하고 적용
  • 웹 접근성 강화: 고대비 색상과 명확한 텍스트 구조로 시각적 가독성 향상
  • 반응형 UI 설계: PC, 태블릿, 모바일 환경에서 동일한 사용성 유지
  • 공공기관에 적합한 신뢰감: 안정적이고 명료한 레이아웃과 컬러 사용
  • 다크모드 적용: 시각 피로 감소 및 접근성 개선을 위한 색상 대응 설계

인터파크 티켓 리뉴얼

본 프로젝트는 팀 단위로 진행한 티켓 예매 사이트 리뉴얼 작업으로,
UX/UI 개선과 반응형 설계를 중심으로 기획, 디자인, 퍼블리싱을 수행했습니다.

  • 고윤정 팀장 정연희, 고경현, 윤다경

    • 인터파크 티켓은 공연, 영화, 스포츠 등 다양한 문화 콘텐츠 예매 서비스를 제공하는 플랫폼입니다.
    사용자 중심의 예매 흐름과 직관적 정보 구조 강화를 목표로 리디자인을 진행했습니다.

    리뉴얼 목표:

    • - 직관적이고 간결한 예매 경험 제공
    • - 반응형 UI로 다양한 기기 최적화
    • - 핵심 기능(검색, 예매, 좋아요) 접근성 강화

    📌 레이아웃

    • Header 페이지 상단에 위치한 로고, 내비게이션, 검색창 등으로 구성된 영역.
      사용자가 주요 메뉴에 빠르게 접근할 수 있도록 구성함.
    • Footer PC에서는 하단 고정, 반응형에서는 중앙 하단에 고정되도록 설정함.
    • banner 새로고침 시 배경색과 콘텐츠가 랜덤으로 변경되며,
      '오늘 하루 열지 않기'와 닫기 기능이 포함됨.

    📌 메인 페이지

    • Section 1 – 메인 비주얼 주요 공연을 배너와 썸네일 슬라이드로 구성함.
      마우스오버 시 배경 이미지 변경되는 시각 효과 적용, 3개씩 슬라이드 정렬.
    • Section 2 – 티켓 오픈 예매 가능한 공연을 슬라이드로 구성하고,
      클릭 시 예매 페이지로 이동되도록 연결함. 반응형 대응 포함.
    • Section 3 – 장르별 랭킹 장르별 탭으로 인기 공연을 전환 표시함.
    • Section 4 – 오늘의 추천 자동 슬라이드 또는 호버 시 배경색과 콘텐츠가 변경됨.
      모바일에서는 리스트와 포스터가 세로로 정렬되도록 구성함.
    • Section 5 – Interpark VIP VIP 할인 공연을 소개하며, 타이틀에 보라색 강조 컬러 적용.
      공연명·가격이 길어져도 레이아웃이 유지되도록 처리함.
    • Section 6 – 인터파크 PLAY 공연 홍보 콘텐츠를 탭으로 구분하고, 탭 클릭 시 관련 영상 슬라이드 전환.
    • Section 7 – 베스트 관람후기 후기 콘텐츠를 슬라이드로 구성, 반응형 적용.
    • Section 8 – 배너 인기 공연 정보를 슬라이드로 정렬하여 시각적으로 배치함.

    📌 서브 페이지

    • 서브페이지 1 – 예매하기 달력, 좌석 선택, 관심 작품 추가 등 사용자 흐름에 맞춘 단계별 UI 설계
    • 1. 예매하기 팝업 진입 시 자동 표시되는 팝업으로 공연 관람에 필요한 정보 안내
    • 2. 공연 상세정보 장소, 기간, 시간, 관심 수 등 공연별 상세 정보 구성
    • 3. 출연진 공연별 출연 배우 정보를 슬라이드로 제공
    • 4. 탭 메뉴 항목별 콘텐츠를 탭 전환으로 제공하여 정보 탐색 효율성 강화
    • 서브페이지 2 – 공연장 공연장별 위치, 대표 공연, 이벤트 등 콘텐츠 구성
    • 1. 공연장 정보 위치 및 공연장별 탭 구성으로 정보 정리
    • 2. 대표 공연 각 공연장에 해당하는 공연을 슬라이드 형태로 시각화
    • 3. BEST 기획 기획전·이벤트 배너 구성, 자동 전환 및 마우스오버 반응 적용
    • 4. 공연 둘러보기 진행 중인 공연을 슬라이드로 구성해 한눈에 볼 수 있도록 설계

    디자인 방향성

    • 직관적 정보 구조: 예매 흐름을 중심으로 한 사용자 동선 최적화
    • 반응형 설계: 모바일~PC 환경 일관성 유지
    • 시각적 일관성: 컬러, 폰트, 컴포넌트의 통일성
    • 사용성 강화: 검색, 좋아요, 로그인 등의 상호작용 단순화

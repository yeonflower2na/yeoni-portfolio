Yeoni's Portfolio

본 프로젝트는 기획부터 디자인, 퍼블리싱까지 모든 과정을 단독으로 수행한 개인 포트폴리오입니다.
편집디자이너에서 UX/UI 디자이너 & 퍼블리셔로 전환한 경험을 바탕으로, 브랜드 아이덴티티와 사용자 경험을 시각적·기술적으로 설계했습니다.
"From Designer to Deblisher"라는 주제를 시각적·기술적으로 구현하여,
브랜드 정체성과 사용자 몰입 경험을 동시에 전달하고자 했습니다.

주요 기능

📌 메인 페이지

  • Intro 텍스트 애니메이션과 3D D 포인트로 주제 전달
  • 3D D GSAP과 Three.js를 활용한 입체적 알파벳 애니메이션 및 인터랙션
  • Preview 프로젝트/디자인 페이지 연결 미리보기 섹션 구성

📌 About 페이지

  • About Me 전환 계기와 경력을 소개하는 모달 구조
  • Skill Bar 퍼센트 기반 숙련도 표시 및 활용 사례 설명

📌 Project 페이지

  • 카드 UI 프로젝트별 카드 디자인 및 상세 페이지 연결
  • 한국소비자원 정보구조 개선, 다크모드 적용, 절차 안내 UI 설계
  • 인터파크 티켓 로그인, 예매 흐름, 공연장 UI 구현

📌 Design 페이지

  • 카테고리 탭 편집·포스터 등 디자인 카테고리별 필터링 기능
  • 호버 효과 작업물 설명 노출로 UX 보완

📌 공통 기능

  • Header 섹션 이동 및 동적 텍스트 변환 기능
  • Footer 연락처 및 HOME 버튼 제공
  • Contact 고정 버튼 클릭 시 정보 모달 노출

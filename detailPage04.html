Journee - 국내 여행 추천 웹앱

본 프로젝트는 기획, UX/UI 설계, 디자인, 퍼블리싱까지 팀이 전 과정을 함께 수행한 협업 프로젝트입니다.
사용자 중심의 흐름을 기반으로 여행지 추천부터 일정 저장까지의 전체 경험을 직접 설계하고 구현했습니다.

  • Shift Left 팀장 김재승, 정연희, 전채아, 정소연, 최시환, 최정민


    • Journee는 사용자가 간편하게 여행지를 추천받고, 일정을 계획 및 저장할 수 있도록 돕는 웹 애플리케이션입니다.
    UI/UX 디자이너로서 사용자 여정을 설계하고, 디자인 시스템 기반의 일관된 인터페이스와 반응형 구조를 구현하는 데 집중했습니다.

    프로젝트는 다음의 목표를 중심으로 진행되었습니다.

    • - 사용자 경험 기반 정보 설계
    • - 디자인 시스템 기반 UI 구성
    • - 반응형 웹 및 접근성 대응

    📌 핵심 기능

    • 로그인 / 회원가입 JWT 기반 인증과 사용자 피드백 중심의 입력 UI 퍼블리싱
    • 여행지 룰렛 추천 랜덤 추천 인터랙션 설계 및 시각적 몰입을 위한 구성
    • 여행 일정 저장 선택한 여행지를 기반으로 개인 일정을 저장하고 조회하는 기능
    • KakaoMap 장소 검색 사용자 위치 기반 장소 검색과 지도 결과 연동 UI 설계
    • 반응형 설계 모바일, 태블릿, 데스크톱 환경에서 동일한 사용성 제공

    📌 디자인 및 퍼블리싱 요소

    • 디자인 시스템 구축 컬러, 폰트, 버튼 등 UI 컴포넌트의 기준을 정의하고 일관성 있게 적용
    • UX 플로우 기반 설계 사용자 행동 흐름을 기반으로 한 인터페이스 구조 및 시각 구성

    디자인 방향성

    • 사용자 중심 정보 흐름: 여행 추천 → 일정 구성 → 저장으로 이어지는 구조 설계
    • 디자인 시스템 구축: 시각적 통일성과 재사용성을 고려한 UI 컴포넌트 정의
    • 접근성 대응: 명도 대비, 폰트 크기, 터치 영역 등 WCAG 기준 반영
    • 반응형 UI: 다양한 기기에서 동일한 시각 경험 제공
    • 감성 UI 구현: 여행의 설렘을 전달하는 색상과 일러스트 사용

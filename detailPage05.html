Class101 클론코딩 프로젝트

본 프로젝트는 Class101 웹사이트를 기준으로 한 클론코딩 프로젝트로,
UI/UX 구조 분석부터 디자인 구현, 퍼블리싱까지 전 과정을 단독으로 수행하였습니다.



실제 서비스의 사용자 경험을 분석하고, 구조와 스타일을 재현하며
퍼블리셔로서의 HTML/CSS 구조 설계 능력과 반응형 구현 역량을 향상시키는 데 초점을 맞춘 작업입니다.

프로젝트는 다음의 목표를 중심으로 진행되었습니다.

  • - 실무 환경에 가까운 웹 퍼블리싱 구조 설계
  • - Class101 UI/UX의 구성 방식 분석 및 구현
  • - 다양한 디바이스에서 안정적인 반응형 UI 적용

프로젝트 바로가기

  • 🔗 클론코딩 결과물 [라이브 사이트]
  • 💻 프로젝트 코드 [GitHub]

제작 일정

  • 분석 2024.11.01 ~ 11.03 : 레이아웃 구조 및 UI 요소 파악
  • 퍼블리싱 2024.11.04 ~ 11.12 : HTML/CSS 마크업 및 반응형 스타일 구현

    • 주요 구현 요소

    📌 메인 페이지

    • 헤더 / 네비게이션 실제 사이트와 유사한 상단 메뉴 구성, 반응형 대응
    • 클래스 카드 UI 이미지, 가격, 좋아요 버튼 등 컴포넌트 구조 구현
    • 배너 영역 그리드 기반 레이아웃으로 시각적 집중도 강화
    • 푸터 실제 서비스에 맞춘 고객지원 및 정보 메뉴 구성

    📌 퍼블리싱 특징

    • 모듈화된 CSS 클래스 네이밍 규칙에 따라 코드 가독성과 유지보수 고려
    • 반응형 웹 구현 데스크톱, 태블릿, 모바일에 따라 유연하게 구조 조정

    퍼블리싱 방향성

    • 레이아웃 중심 설계: 실제 서비스에 기반한 구조 분석 및 그리드 구성
    • 스타일 일관성: 텍스트, 버튼, 카드 UI의 디자인 톤 통일
    • 미디어쿼리 적용: 뷰포트에 따른 콘텐츠 재배치 및 유연한 대응
    • 접근성 고려: 시멘틱 태그 사용 및 시각 흐름 정리
    • 코드 구조 정돈: 반복 가능한 컴포넌트 중심의 CSS 설계
